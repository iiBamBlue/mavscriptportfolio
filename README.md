## Deploy to GitHub Pages

1. Build the project:

   ```bash
   npm run build
   ```

2. Install the `gh-pages` package:

   ```bash
   npm install --save-dev gh-pages
   ```

3. Add the following scripts to your `package.json`:

   ```json
   "scripts": {
     "predeploy": "npm run build",
       "deploy": "gh-pages -d dist"
     // ...other scripts...
   }
   ```

4. Deploy the project:

   ```bash
   npm run deploy
   ```
